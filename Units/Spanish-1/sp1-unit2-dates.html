📝 Grammar
Date Formula
Hoy es + day + el + number + de + month. Example: Hoy es lunes, el quince de octubre.
El Primero
Use el primero for the 1st of the month. Use cardinal numbers (dos, tres...) for all other dates.

📝 Grammar
All 12 Months
Months are NOT capitalized in Spanish. They follow the same pattern: el + number + de + month.
Future Dates
El examen es el viernes, el tres de mayo. — The exam is Friday, May 3rd. Use the same formula for any date.
